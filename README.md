# Alleria

> 本项目正在开发阶段

Alleria 是一个基于 Django 的管理后台框架，目前包含以下功能：
* 用户管理
* 角色权限管理
* 部门管理
* 菜单管理
* 数据字典管理
* 日志 / 全文搜索

技术特点：
* Django / Django channels
* Bootstrap 3
